# css选择器

## 选择器


### 后代选择器

空格隔开 
```js
.one p{
    
}
```
### 子代选择器

挨着的下一级【>】
```js
.one > p{
    
}
```

### 兄弟选择器 

相邻兄弟选择器：【+】 挨着
```js
.one + p{
    
}
```
普遍兄弟选择器：【～】同级
```js
.one ~ p{
    
}
```

### 选择器组

交集选择器，精准悬着
```js
// 即是div 又有为box的class 【要挨着一起】
div.box{
    
}
```

并集选择器

```js
body, p, h1, h2{
    
}
```

### 伪类

某个元素处于某种特殊的状态❄️
- 动态伪类
- 目标伪类
- 语言伪类
- 元素状态伪类
- 结构伪类
- 否定伪类


```js
div:hover{
    
}
```
